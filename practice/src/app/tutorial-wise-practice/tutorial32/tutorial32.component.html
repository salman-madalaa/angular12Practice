    Why videos TS in angular tutorial ? :
    when ever we are working with angular we are susing core javascript.
    With the type script we can provide types to our functions to our properties .
    Ex : data :number = 20 here data must be number only so the it can not access string .

    Basic Types :
    Here are some basic types like number, string, boolean, any... etc
    And we have assign two type also Ex: data: number | string then data will access only these two types .
    This can write inside the function parameters also working Ex: getData(item:number | string) like this .
    if you assign value to the variable ,then type script defaulty assign the type based on the given value
    Ex : item = 20 the item type number is declared automatically . when you can hover the cursor on item it will show item:number .
    Note : any is never recommended . Because it decreases the power of TypeScript .

    Type with object and array :
    Syntax : Object : Ex: " obj : [name:string,phn:number] = [name:"salman",phn:123] "
    Note : inplace of '[]' please put 'flower brackets' , here i put [] for removing error for to define syntax .
    Array : arr :number[]=[10,20,30,40,50] so this array accept only numbers .
    for string array syntax : arr : string[]=["salman","visesh","vijay","prasad"]

    Suppose If we want to assign two typs array in a same array that is also possible .
    Syntax : Ex : arr2 : (string | number )[]=["salman",20,"vijay",30]
    So that we can pass two types of values in a same array arr2 based on the above syntax .

    Interview Questations :
    1) can we use Type script only in angular ?
    A) No , wecan use Type script in react ,vue, also .
    2) Is there any difference between Type script and JavaScript ?
    A) yes , Type script is the extended version of javascript. Here we can define class and types of propertiesand objects and array .Here we can also define the interface,inheritence,enums...etc
    3) can a browser compile the type script ?
    A) No, in angular in build type script compiler , this compiler convert the type script code into javascript .Then it will run on our browser .
