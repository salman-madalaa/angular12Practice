    Make child component:
    For to create child component we have to run the command ng g c child

    Use child component:
    For to use child component you can add that selector in the parent component .
    Suppose if you are write the child selector inside the app.component.html at that time app.component.html is the parent .
    Now I am using the child selector inside the tutorial27.component.html file so this file is the parent .

    Send data from parent component:
    If we send data from parent to child we can give that data inside the child selector which is present in the parent .
    Ex : For example now I send data to the child component from tutorial27.component.html ,i give the data inside the child selector([item]="data") which is present in the tutorial27.component.html . Here item is the variable you can write any name .
    Next we have to go child.component.ts file and give input decorator like @Input() item . Here name should be same as we give in the child selector which is present in parent tutorial27.component.html .
