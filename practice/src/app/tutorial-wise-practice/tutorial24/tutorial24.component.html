    What is Bootstrap :
    Bootstrap is basically a library that makes our application responsive .
    Rsponsive means an application that can working fine all kinds of devices like mobile,tablets,desktop,laptops....etc

    How to Install Bootstrap :
    If we have to install the bootstrap there are two ways 1) command Line 2) adding cdn link in index.html file
    Through the command line we have to add bootstrap with two ways a) npm command b) ng command . But ng command is recommended . Because if you use the npm command you have to make some manual changes . If you use ng command you don't need any manual changes .
    Commands : npm install bootstrap/latest ;
    ng add @ng-bootstrap/ng-bootstrap (this is recommended) ;
    This command will install two packages related to Bootstrap first one is @ng-bootstrap and second one is ng-bootstrap

    Check Changes after instal bootstrap :
    After run the above command there is some changes are happend in our projects files . Those are
    1) angular.json : bootstrap.css is added in our Style Property like " node_modules/bootstrap/dist/css/bootstrap.min.css "
    2) package.json : we can see the 3 packages are installed
    a) @ng-bootstrap/ng-bootstrap": "^10.0.0" : that makes bootstrap comfortable with the angular .
    b) "bootstrap": "^5.1.3" : It has the Core Features of bootstrap like responsive,datepickers,buttons,alerts...etc .
    c) "@angular/localize": "~12.2.0" : when we run the application on different different countries and languages that time we want to sport multiple languages in our system. This package is will help make multiple languages supported in our application .     3) app.module.ts : ngbModule is imported from ng-bootstrap module and registered inside the imports
    4) polyfills.ts : this file is also make a depedency with the localize init
