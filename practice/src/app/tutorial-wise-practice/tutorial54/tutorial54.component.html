    {{'why we are need lazy loading' | titlecase}} :
    --> When app component is load all the components is loading at a time . This not good practice because with out using the component we are loading that route.
    --> So angular introduced lazy loading . Based on this lazy loading we can load the route at thr time of required . so that app module does not load all the components at a same time.

    {{'make a module and components' | titlecase}} :
    --> Here i create a two modules admin1,user1 modules by commad line : ng g m admin1 --routing , ng g m user1 --routing , --> And creating two components in each module .

    {{'Use module in lazy loading way' | titlecase}} :
    Note : When we use lazy loading we did not import the module in app.module.ts in imports section
    Synatax: path:'admin',loadChildren:()=>(./admin/admin.module).then(module=>module.AdminModule)
