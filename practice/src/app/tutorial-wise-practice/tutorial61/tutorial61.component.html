    • {{'Angular 13 Features' | titlecase}}


    {{'TypeScript 4.4 Support Module' | titlecase}} :
    --> TypeScript 4.4 support has been added;as of October 5,
    --> Versions older than TypeScript 4.4.2 are no longer supported in the core


    {{'View engine Removed' | titlecase}} :
    --> In angular 13 View engine is completely removed
    --> It is replaced with angular Ivy engine
    --> Angular next-generation compilation and rendering pipline .
    --> So some of the extra code from angular is removed and angular has become littlebit light weight .

    {{'Node js Support.' | titlecase}} :
    --> Node.js versions older than v12.20 are no longer supported .
    --> The current Node js version 16.13.0
    --> You may face issues to install packages with older versions .

    {{'Angular CLI improvements' | titlecase}} :
    --> Angular now supports the use of persistent build cache .
    --> We can tur on and off build cache by angular.json file .
    --> With cache option speed of build is increase up to 68%

    {{'Rxjs version updated' | titlecase}}
    --> RxJS 7.4 is now the default for apps created with ng new .
    --> Existing apps using RxJS v6.x will have to manually update using the npm install rsjs@7.4

    {{'Validation Improvements' | titlecase }} :
    --> Now we can enable and disable validations like min, max,email etc .

    {{'IE 11 Support' | titlecase}}
    --> From now angular dont support IE 11
    --> So some polyfil files also removed for IE 11 and angular become more faster

    {{'Some other changes' | titlecase}}
    --> Improvements to TestBed enable a better job of tearing down test modules .
    --> $localize is a stable API, for internationalization
    --> Vaalidation Error messaging has been improved .

    Install Angular 13 --> npm update angular-cli-g
    or
    npm uninstall -g @angular/cli
