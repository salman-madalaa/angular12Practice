    Install Material UI: 1) What is Material UI?
    A) This is the html and css library. That can enhance the application look and feel .
    2) how it can enhance ?
    A) It can add some extra properties to our buttons,input fields,datepickers, ..etc . And they will look more beautiful then the default style .
    Installation :
    Run the command : ng add @angular/material , it will ask some questations give 'yes' to all the questations.

    Check changes after install material UI :
    After you run the above command you can see some changes occer in our project files
    Those files are :
    1) angular.json : here it can add the styles of material ui from node_modules in styles section
    2) style.css : it add some basic styles in globally 3) package.json : it will add the dependencis like @angular/cdk and @angular/material 4) index.html : it add 3 links for font styles 5) app.module.ts : in this file it import the BrowserAnimationModule and register in imports section
