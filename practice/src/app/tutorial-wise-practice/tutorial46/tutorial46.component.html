    {{'what is service ?' | titlecase}} :
    Services are the basically classes that may have some data,property,functions . And we can use this data ,property,function in different files .
    For Exampe if we have two pages home page and user page ,and you wan to share the data between two components that time we can use the service.

    {{'why we need service?' | titlecase}} :
    --> give same data on the two pages .
    --> For example we want to get the user list on home we will call the API on the home page or second thing same API we should call on the user page
    --> Here we are actually duplicating the code . Because same API call from two pages .
    --> So that time we create a service and call our API on that service and share the data between the two components.
    --> We can simply say that the Services are the Helpers . that can share the data between multiple components .

    {{'understand the problem statement' | titlecase}} :
    • {{user.name}} -- {{user.phoneNumber}}--{{user.address}}
    --> If we want to display above information in another component we have to copy the userData of .ts file and paste in another .ts file
    --> And this is not good practice because we are duplicating the code .
    --> So at this time we need the service .
