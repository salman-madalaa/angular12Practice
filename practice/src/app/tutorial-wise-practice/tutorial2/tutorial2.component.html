    {{'How to install Node and NPM' | titlecase}} :
    --> First we have to go to node js website and install node js download link
    --> After download execute the exe file then you can easily install the node js in our system
    --> After finish just type the command node -v so that you can get the version of the node js .

    {{'How to install angular CLI' | titlecase}} :
    --> if we want ti install the angular cli use the command npm install -g @angular/cli . here -g meand globally installed .
    --> when you run the above command you can get the angular cli download automatically into our system .
    --> for to check the angular cli version ng version.

    {{'Install First Agular app' | titlecase}} :
    --> For to create a app type the command ng new projectName (or) ng n projectName
    --> If you run the above command it will ask some questations give yes/no type then you give that options.
    --> After create the project go to the project folder and run the command ng serve.
    --> After run the above command you can get the app is running on port number 4200
    --> Go to browser and type http://localhost:4200 then you can get the out put of the project .
