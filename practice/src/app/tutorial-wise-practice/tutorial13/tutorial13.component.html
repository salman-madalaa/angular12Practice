    In Angular there are Four Types of Styles are present :
    1)Inline Style : (With In the Tag Ex: style= "font-size:20px" ) = Inline Style
    2)Internal Style : (With In the component ".html" file Ex: style /style ) = This is Internal Style Example
    3)Component Style : (With In the component ".css" file) = This is Component Style Example
    4)Global Style : (With In the "style.css" file This is Applied All over the App ) = This is Global Style Example
    Note : If we apply same styles to multiple components at that time we use Global Style

    Note: If we give more then one Style the Priority Wise
    first priority given to Inline Styles --> Internal Styles --> component Styles --> At last go to Global Styles
