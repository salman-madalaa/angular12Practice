    Make A Simple Form :
    --> In Reactive Form most of the code will be in class file that is .ts file .
    Note: If you want to use the reactive forms you must have to import the ReactiveFormsModule in the module.ts file
    --> Go to module.ts file go to import section and import ReactiveFormsModule
    --> After import the ReactiveFormsModule , go to the component.ts file and import two more imports FormGroup,FormControl

    Connect Form controls with Form :
    --> FormGroup : It will control our complete form. And it will handle that what we are getting inside the complete form and this is valid or not .
    --> FormControl : it will specificly handle a single input field.
    --> Next we have to write studentForm = new FormGroup( name:new FormControl(''), email:new FormControl(''), phoneNumber:new FormControl(''), address:new FormControl(''), marks:new FormControl(''), ) inside the .ts file.
    -->Next we bind that the studentForm in the form tag like [FormGroup]="studentForm" . So that we can get all the values from the form . --> When we use formGroup in form tag we must have to specify the name like formControlName =" " to the every input field .
    Note : studentForm which is the value of the FormGroup should be match with the .ts file name . And formControlName should be match with the names which we give inside the formGroup in .ts file
