    {{'make module' | titlecase}} :
    --> for to make module use the command ng g m moduleName

    {{'component inside module' | titlecase}} :
    --> After creating a module next run the command for creating component ng g c componentName
    --> After creating component use exports section inside the module.ts file and export that componets in to app module
    --> Next import that new creating module in to app.module.ts file in import section. so that we can acces that that module entire of application.

    {{'show module component' | titlecase}} :
    --> For show the componet use the componet selector

    {{'Add routing and routing link' | titlecase}} :
    --> Go to the app-routing.module.ts and import the two componets inside that file .
    --> Give the path:'' and component: ComponentName in routes section . So that we can go that component based on out path .
    -->next use the routerLink property for to define the route Syntax: routerLink="pathName"
    --> Use router-outlet tag for displaying the components .
