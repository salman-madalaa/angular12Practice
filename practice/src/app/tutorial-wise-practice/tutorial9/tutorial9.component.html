    {{'What is Module ?' | titlecase}} :
    --> Module is a compleate feature of your application .
    --> For example we have to implement authentication so that we can create login,logout,reset,...ete components .
    --> All these components we can achieve authentication feature .

    {{'How we can make a module' | titlecase}} :
    --> For Create a module we run the command ng generate module moduleName (or) ng g m moduleName
    --> For example create a authentication module we can run the command ng g m authentication
    --> Then you can get the authentication folder inside that we have one file " authentication.module.ts " .
    Note : After create module import that module inside app.module.ts file import section ,then it will work .

    {{'how to use module' | titlecase}} :
    --> In this tutorial i create a TutorialWisePracticeModule in that module i create all tutorials concepts .
    --> After create that module i import that module inside the root module(app.module.ts) in imports section.
