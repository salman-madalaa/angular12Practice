    Use of pipes :
    Pipes are basically use for transfor the data from one formate to another formate . for example you have a string and you want to make it uppercase/lowercase you can use pipes . update the date formate ,and you wan to apply the percentage or number ....

    Problems that can resolve with pipes :
    If we want to chnage the formate of text like uppercase,lowercase we don't need to write special code .
    In angular in built pipes are available . Syntax : " | "
    Ex: {{ 'salman' | uppercase }} Here salman is the lowercase text convert into uppercase .
    In angular there are different types of pipes are present :
    uppercase, lowercase, date, date: 'fullDate', date: 'shortTime', date: 'd/M/y', currency, DecimalPipe, json, percent ........etc

    Example :
    UpperCase Pipe :
    {{ 'cassiniplm' | uppercase }} = this is for uppercase
    LowerCase Pipe :
    {{ 'CASSINIPLM' | lowercase}} = this is for lowercase
    Date Pipe :
    {{today}} {{ today | date: 'fullDate' }} = this is for full Date
    {{ today | date: 'shortTime' }} = this is for short Date
    {{ today | date: 'dd/MM/yyyy' }} = this is for date format based on the format

    1) is it mandatory to use only pipes to make lowercase and uppercase formate?
    A) No , we can also make out own custome functions .
    Ex : let make the function capString() and it have a parameter item type of string in .ts file . And we call that fuction from html file then we can get the result .
