    Note : Angular for loop and java script for both are the different . When ever we have to make any loop in side .html file at that time we use angular for loop , but when ever we have to make any loop in side .ts file that time we use java script for loop


    Make Array :

    Syntax: users = ['anil', 'salman', 'prasad', 'vijay', 'visesh'] in .ts file


    Apply for Loop on it :

    Syntax: *ngFor = "let user of users"


    Displaying the users :

  • Name is : {{user}}

    • Make Array of object :

    Syntax: userDetails = [ doubleflowerbrackets, doubleflowerbrackets, doubleflowerbrackets] in .ts file

    Inside the flower brackets we have to define = name:'salman',address:'hyderbad',email:'madalasalman@gmai.com' ..... like this

    Apply For loop on userDetails Object Array :

    Syntax: *ngFor = "let userDetail of userDetails" in tr tag which is inside the tbody

    Displaying the usersDetails :

    Name Email Address
    {{userDetail.name}} {{userDetail.email}} {{userDetail.address}}
