    What is two way binding :
    Two way binding simply means updating and displaying the property at the same time .
    For example if you have input field when ever you want to type something inside it you want it that property will be update and it will display on the screen .

    Make text input and define the property :
    Syntax : [(ngModel)] = "value" .
    Note : If you want to use two way binding you must have to import FormsModule inside the app.module.ts file. {{value}}
