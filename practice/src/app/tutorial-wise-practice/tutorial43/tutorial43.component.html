    What is Dynamic Routing :
    Dynamic routing, also called adaptive routing, is a process where a router can forward data via a different route or given destination based on the current conditions of the communication

    How to pass data in the routing :
    --> For to pass data to a routing fir the route have the extension /:id .
    --> When you pass that id in the router we have to provide the id in the routerLink property . Syntax : routerLink="student/2"

    How to get data from routing :
    --> For to get data from router we import the ActivateRoute module in the .ts file .
    --> For to get the id from the route route.snapshot.paramsMap.get("id") write this in component.ts file.
    Note : Here Name should be same in path and the route.snapshot.paramsMap.get("id") .
    Suppose if we get this page id the code is in tutorial43.component.ts file . in your console or visual studio code .
