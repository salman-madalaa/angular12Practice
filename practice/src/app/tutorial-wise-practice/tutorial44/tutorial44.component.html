    {{'What is 404 page' | titlecase}} :
    When a visitor hit a n invalid url in a website or in our web application that time we say it this is the 404 error

    {{'Make Component for 404 Page' | titlecase}} :
    --> For to make a 404 page component we have to create the component like page-not-found Syntax: ng g c pageNotFound

    {{'Add Routing For 404 page' | titlecase}} :
    --> When enter wrong url to redirect 404 page add this component in app-routing.module.ts file , and give the path "**"
    Syntax: path:'**'',Component:PageNotFoundComponent . (Here ** means universal route that time we call page not found component.)
    Note : This must be put at the end of the all routes.
