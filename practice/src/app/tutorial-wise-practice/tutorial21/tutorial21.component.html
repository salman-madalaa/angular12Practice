    Make Header Component :
    1) First you create the header component from command line by using " ng g c header " this command
    2) After that you can add this in root module that is app.module.ts file (when you run the above command by default this step is done automatically).
    3) And next you should use header component selector(app-header) inside the any component then you want the header component at that module. Note : In this app we already use the header component so check the code .
