    {{'what is routing module' | titlecase}} :
    --> When we define the routing inside the module then we called a routing module .
    When we have a small application we can take whole routing inside app-routing.module.ts file
    This is not possible for big application so we have to create routingmodule individually.And that is the best practice.

    {{'make module with routing file' | titlecase}} :
    --> For to create Routing Syntax: ng g m moduleName --routing
    Note : After creating routermodule make sure that new rouring module should be import in root routing module (app-routing.module.ts) file in imports section

    {{'Define Routing inside module' | titlecase}} :
    --> Next Import the new module inside the app.module.ts of imports section .

    {{'make routing link routing link' | titlecase}} :
    --> Go to the app-routing.module.ts and import the componets inside that file .
    --> Give the path:'' and component: ComponentName in routes section . So that we can go that component based on out path .
    -->next use the routerLink property for to define the route Syntax: routerLink="pathName"
    --> Use router-outlet tag for displaying the components .
