    Make New App:
    --> For to make new App run the command ng n AppName .
    --> When you run the above command command line asking when routing is add or not ? . So you write yes and click enter.
    --> So that the command line automatically include the routing to our app .
    --> In our app folder we can get the one more file app-routing.module.ts in our app folder.

    Make Component :
    --> for to make component we must run the command ng g c componentName .
    --> so for routing we have to create 2 or 3 components .
    Note: I already created components in this tutorial so i did not create the dummy components .

    Apply Routing :
    --> In routing.module.ts file we have to give path and component name path:'home',component:HomeComponent .
    --> For to use the routing we have to write " router-outlet " tag inside the html tag .
    --> Through the router-outlet we can get the allcoponents on that html page .

    Interview Questations :
    1) How we can pass the id to the router?
    A) For to pass the id to the router we have to give the ' /:id ' in the router path . For to give the id we give like routerLink="student/1"
