    {{'what is interrpolation ?' | titlecase}} :
    --> Interpolation is basically use display the dynamic data in our html page .
    --> Dynamic data is nothing but define data inside the component.ts file and display that data in the html file using interpolation.
    --> interpolation Syntax is doubleflowerbrackets

    {{'How to use it ?' | titlecase}} :
    --> Here I take an example in component.ts file declared temp = cassiniPLM and display that temp on the screen .
    Result : {{temp}} .

    {{'What we can do with the Interpolation ?' | titlecase}} :

    Result : {{temp}} .
    --> Even we can get the data of a function also .
    Ex: {{add()}} ; {{getValue()}}
    {{ 2+2+2*20}}
    {{num == 100}} Note : So we can achieve get the function values , true or false, addition, multiplication .... etc

    {{'what we can not do with the Interpolation' | titlecase}} :
    --> we can not to the assign values inside the interpolation like num = 200 .
    --> we can not use typeof inside the interpolation .
    --> We can not use new keyword .
    --> you can not add incremental values like ++,--
