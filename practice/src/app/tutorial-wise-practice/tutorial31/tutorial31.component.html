    Use of template variable :
    1) What is the use of template Ref variable ?.
    A) we have to get property of input box in two way a) by using DOM like id b) template ref variable
    Suppose if we want to get all the properties of input field easily by using the template ref variable

    Make input type and Use Template Ref variable :
    Syntax : #box
    input box: placeholder : {{box.placeholder}}
    name : {{box.name}}
    value : {{boxValue}}

    In the above example we have to get the properties of the input box using template ref variable
    when we enter text input box after click on click me button then you can get the value what you enter in the input box .
