    Params with pipes :
    In pipes we also pass the paramaeters for get some periculer string in a string text .
    For Example in 'cassiniplm' we want only 'plm' string then at that time we have to pass the parameters .
    {{ 'cassinplm' | slice:6}}
    Syntax : " | slice:6 "

    Use 2 pipes at the same time :
    {{ 'cassinplm' | slice:6 | uppercase}}
    In the above example we use two pipes at the same time . we pass the 'cassiniplm' string through the "slice:6 " pipe then we can get 'plm', and pass that 'plm' text through the uppercase pipe it convert to uppercase .

    In angular there are different types of pipes are present :
    uppercase, lowercase, date, date: 'fullDate', date: 'shortTime', date: 'd/M/y', currency, DecimalPipe, json, percent ........etc

    Example :
    UpperCase Pipe :
    {{ 'cassiniplm' | uppercase }} = this is for uppercase
    LowerCase Pipe :
    {{ 'CASSINIPLM' | lowercase}} = this is for lowercase
    Date Pipe :
    {{today}} {{ today | date: 'fullDate' }} = this is for full Date
    {{ today | date: 'shortTime' }} = this is for short Date
    {{ today | date: 'dd/MM/yyyy' }} = this is for date format based on the format
    Currency Pipe :
    {{ 1234.56 | currency:"CAD" }}
    {{ 1234.56 | currency:'CAD':'symbol-narrow'}} = this is for currency
    Decimal Pipe :
    {{ 3.14159265 | number: '3.1-2' }} = this is for decimal before the dot(.) 3 numbers, after the (.) dot we have 2 digits
    {{ 3.14159265 | number: '1.4-4' }} = this is for decimal after the (.) dot we have 4 digits .
    Json Pipe :
    with out pipe : {{ jsonVal }}
    with one pipe : {{ jsonVal | json}}
    with two pipes : {{ jsonVal | json | uppercase}}
    Percent Pipe :
    {{ 0.123456 | percent }}
    {{ 0.123456 | percent: '2.1-2' }}
    {{ 0.123456 | percent : "10.4-4" }}
    Slice Pipe :
    {{ [1,2,3,4,5,6] | slice:1:4 }}
    {{ [1,2,3,4,5,6] | slice:2 }}
    {{ [1,2,3,4,5,6] | slice:2:-2 }}
