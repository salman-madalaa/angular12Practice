    {{'What is Component ?' | titlecase}} :
    --> Components are the building blocks of our application . That are used to develop any specific feature or functionality .
    --> For example if we want to make a header in our website that time we can create a component and write your header code inside that commponent .
    --> when ever we make a component we can easily reuse it at any time .
    --> we can create component at one time and reuse it many times.

    {{'How we can make a component' | titlecase}} :
    --> For Create a component we run the command ng generate component componentName (or) ng g c ComponentName
    --> For example create a login component we can run the command ng g c login
    --> Then you can get the login folder inside that folder we have four files " css, html, spec.ts, ts " files .

    {{'how to use Component' | titlecase}} :
    --> if we want to use one component in another component we have use the component Selector
    --> Component selector is present in the .ts file
    --> use that selector inside any other component you can see the out of that selector .
    --> for example login copmonent we cann see the "app-login" is the selector which is present in the login.component.ts file .
